# Module 16 — Cheatsheets

## What You Will Find

Short, table-based quick references for the commands you'll use most. Keep these open while practicing the labs and projects.

## Why This Module Matters

Nobody memorizes everything. Good engineers look things up fast. These cheatsheets are tuned for speed — command, purpose, example, real-world use.

## How to Use

- Skim the relevant sheet before a lab.
- Don't memorize — recall the *pattern*, look up the *flag*.
- Pair with the full topic files when you need the "why".

## Quick Revision

These sheets distill Modules 02–13 into lookups. If a row is unfamiliar, revisit its source module.
